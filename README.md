# Job Portal Project

A professional job portal built with **PHP, MySQL, Bootstrap 5, and PDO**.  

## Features

### General
- User registration and login (admin, employer, job seeker)  
- Role-based dashboards  
- Secure session management

### Employer
- Create company profile  
- Post jobs  
- View applicants  

### Job Seeker
- Browse available jobs  
- Apply to jobs  
- Track applications  
- Upload CV (optional)

### Admin
- Manage users, companies, jobs  

### UI/UX
- Responsive design using Bootstrap 5  
- Modern color palette and Inter font  
- Alert messages for success/error
